# Diferencias entre Classes y Objects

En Python, las classes y objects trabajan mano a mano para organizar y menejar datos.
Construyes una class para definir comportamientos, entonces creas un objeto que use esos comportaminetos.

En otras palabras, una class es como un plano o una plantilla que usamos para crear objects.

Veamos que son las classes, y como usarlas para crear objetos.

Para crear una class, usamos la palabra clave 'class' seguido de el nombre de la class y dos puntos (colon).
Entonces escribes la class, puedes agregar un inicializador, seguido por cualquier atributo o methodo.

Los atributos son como variables dentro una class, y son usados para guardar datos.
Los methodos son funciones definidas dentro de las class, y son las acciones que pueden realizar los objetos creados con una clase.
