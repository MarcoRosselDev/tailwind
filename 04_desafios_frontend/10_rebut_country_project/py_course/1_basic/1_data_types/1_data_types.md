# 1 que es python y cuales son algunos usos comunes en la industria

Python es un lenguaje de programacion de uso general, conosido por su simplesa y fasilidad de uso.
Esta facilidad de uso convierte a python el lenguaje de programacion mas popular en estos tiempos modernos.

Python es usado en muchos campos como: 

- data science (ciencia de datos) 
- machine learning (aprendisaje automatico)
- web development (desarrollo web)
- scripting (programacion)
- automation (automatizacion)
- embedded systems (sistemas enbebidos)
- y muchos mas...

Python es el lenguaje principal que la mayoria de data scientist y machine learning engineers usan corrientemente.
Librerias como Panda y Numpy hacen que el analicis de datos sea menos tedioso,
mientras que otros como Tensorflow y Scikit hacen machine learning y AI models mucho mas accessibles.

En el desarrollo web, librerias Python como Django, FastAPI, y Flask permiten a los desarrolladores construir sistemas back-end escalable y seguro con minimos esfuersos.
Muchas plataformas de redes sociales como instagram y Pinterest usan Python en el back-end.

Profesionales en ciberseguridad y  ethical hackers usan Python para detectar vulneravilidades como malware y otros virus, construir escaners de seguridad automaticos y analicis de amenazas.

Python corre bien en microcomputadores como Raspberry Pi y placas compapible con MicroPython, entonces tu puedes construir todo tipo de proyectos loT como dispositivos de hogar inteligentes, estaciones de monitoreo de clima, y mas.

Python es ampliamente usado en DevOps para escribir CI/CD scripts y manejo de infraestructura atraves de canales de desarrollo (development pipelines).
Tambien es comunmente usado para construir servicios back-end y APIs internas.

En software testing, herramientas Python tales como pytest son usados para crear conjuntos de pruebas fiables, mientras 

finalmente, unod e los mayorest fortalezas de Python es la automatizacion.
Puedes escribir simples scripts para ayudarte con la repeticion de tareas como extraer datos de spreadsheets(hojas de calculo),
enviando emails, y trabajar con archivos en tu maquina local.

Librerias como Selenium y BeautifulSoup tambien hacen esto facil de interactuar con sitios web, entonces puedes scrape(rascar) data publica, automatizar tareas atraves de una UI web, y incluso manegar cloud deployments (desplieges en la nuve) para tus proyectos.

Como puedes ver, Python es un lenguaje vastante poderoso, y si, es facil de aprender.
Dede simple automatisacion de scripts a grandes escalas, nivel industrial de aplicaciones, tu puedes usar python para cualquier cosa.

Python es una gran opcion para cualquiera que quiera aprender a programar,
independiente de en que quieran especializarce mas tarde.

# 2 typos de Dato en Python y como obtener el tipo de variable ?

Antes de trabajar con variables Python, es importante entender los tipos de datos.
Un tipo de dato describe el tipo de valor que una variable hospeda.
Por ejemplo, un numero, un poco de texto, o una lista de items.
Los lenguajes de programacion usan tipos de datos entonces ellos conocen como almacenar y trabajar con diferentes tipos de informacion.

aqui una lista de los tipos de datos mas comunes que usaras en Python:

- Integer (conjunto de enteros en matematicas)
- Float (punto decimal)
- String (cadena de caracteres)
- Boolean (true o false)
- Set ({9, 0.4, "apple"}) (coleccion desordenada de items)
- Dictionary ({clave, valor} === objetos en js)
- Tuple ("hola", 7 , 9.5) === (lista ordanada inmutable)
- Range (secuencia de numeros , usados en loops)
- List (coleccion ordenada de items, que soporta diferentes tipos de dato)
- None (representa la ausencia de valor == Null en js)


Para tener el tipo de dato en Python utilizamos
type(<data tipe>)
