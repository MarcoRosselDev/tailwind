# 1 que es python y cuales son algunos usos comunes en la industria

Python es un lenguaje de programacion de uso general, conosido por su simplesa y fasilidad de uso.
Esta facilidad de uso convierte a python el lenguaje de programacion mas popular en estos tiempos modernos.

Python es usado en muchos campos como: 

- data science (ciencia de datos) 
- machine learning (aprendisaje automatico)
- web development (desarrollo web)
- scripting (programacion)
- automation (automatizacion)
- embedded systems (sistemas enbebidos)
- y muchos mas...

Python es el lenguaje principal que la mayoria de data scientist y machine learning engineers usan corrientemente.
Librerias como Panda y Numpy hacen que el analicis de datos sea menos tedioso,
mientras que otros como Tensorflow y Scikit hacen machine learning y AI models mucho mas accessibles.

En el desarrollo web, librerias Python como Django, FastAPI, y Flask permiten a los desarrolladores construir sistemas back-end escalable y seguro con minimos esfuersos.
Muchas plataformas de redes sociales como instagram y Pinterest usan Python en el back-end.

Profesionales en ciberseguridad y  ethical hackers usan Python para detectar vulneravilidades como malware y otros virus, construir escaners de seguridad automaticos y analicis de amenazas.
